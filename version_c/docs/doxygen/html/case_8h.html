Enum documentation

enum CaseType_S

all case type

Contains all type that a case can contain. Each case can contain 4 elements types : a part of the production line a resource case an empty case a transdimensional gate

3 resources is produced from a resource case every 10 turns

In order to know if a case contain a box (that contain resources and garbage), we use the first bit of an integer. The case contain a box -> odd number The case does not contain a box -> even number

Enumerators
CASE_VIDE

a case that contain nothing
CASE_GATE

a case that contain the transdimensional gate
CASE_SOURCE

a case that contain resources
CASE_MACHINE

a case that contain a machine
