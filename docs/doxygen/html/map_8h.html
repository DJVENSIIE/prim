  • finish a turn
  • add a machine in a case
  • upgrade a machine, already created in a case

Map* map_create(Difficulty dif)

Create a new map.

Parameters
dif in a valid difficulty chosen by the user
Returns a new map that contains all the map information

In order to begin a new game, we need a board game. It will contains all the map information concerning machines emplacement, sources, resources, garbage, Staff and gate, presents on the map.

Difficulty has been chosen by the player which determine the width and height of the map Creation of the map : a bi dimensional Case table 2 sources cases and 1 transdimensional gate are randomly placed into those cases. The player starts the game with 5 FISE, 5 FISA, 100 E and 100 DD. Initialization of the turn, the score the pollution to 0. By default, the FISA production is E. A table which contain staffs that the user bought is also initialized

ErrorCode map_endTurn(Map* map)

Finish a turn.

Parameters
map game map
Returns an error that specify what is the problem

Verifications to do at the end of a turn

  1. Check FISA and FISE production
  2. Update E

  3. Call all conveyor belt/cross at the same time.

    1. Resources moves
    2. Verify each resource destination
    3. next case empty then resource is lost
    4. next case is an ENTRY machine -> send to machine
  5. Check if we produce resources (sources)
  6. Check if we produce wastes (gate)

  7. Call all Recycling center

    1. stored garbage don't decrease DD
    2. 10 garbage produce 1 /per turn

  9. Call all collectors

    1. activated from top to bot, left to right
    2. No source then do nothing
    3. randomly choose a source among all adjacent ones
  11. Update DD value
